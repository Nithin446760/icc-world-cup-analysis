# ICC Cricket World Cup Analysis

## Project Overview

This project analyzes historical ICC Cricket World Cup ODI matches using Python and Pandas. The objective is to identify team performance trends, winning patterns, tournament competitiveness, and historical dominance across World Cup seasons.

## Dataset

* Historical ICC Cricket World Cup ODI match data
* Includes match results, teams, winners, venues, hosts, and tournament seasons

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
* Git & GitHub

## Analysis Performed

* Data Cleaning
* Missing Value Analysis
* Team Performance Analysis
* Winning Trends
* Match Competitiveness Analysis
* Venue and Host Analysis
* Data Visualization

## Key Insights

* Identified the most successful teams across World Cup history.
* Analyzed win percentages and performance trends.
* Examined how tournament competitiveness changed over time.
* Investigated historical World Cup winners and dominant eras.

* `ICC-WorldCup-Analysis.ipynb` - Main analysis notebook
* `icc-cricket-world-cup-odi.csv` - Dataset
